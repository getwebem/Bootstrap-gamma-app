#Mobile App clone
###**Getting started**
###Technologies Used
####HTML5, CSS3, Bootstrap 3.3, Photoshop
 - Run :  `git clone https://github.com/getwebem/gamma-app.git`
 - Run :  `cd gamma-app`
 - Run :  `Brackets .`
 - Open:  `Index.html`
 - Open:  `Live Preview`
